# ChirpMining
Web application written in PHP to manage Twitter with helper tools. 


You will need a config.php with the following code: 
```
<?php
    // Twitter configuration - you need a developer account
    // https://developer.twitter.com/en
    $access_token = "";
    $access_token_secret = "";
    $consumer_key = "";
    $consumer_secret = "";
?>
```
